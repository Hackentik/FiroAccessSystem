# FiroAccessSystem

**Firo AccessSystem** - открытый проект сетевого сервера СКУДа который был создан просто так.

Система позволяет контроллировать доступом, открывать двери по расписанию и в будущем будет иметь API.
На данный момент система сырая и может быть куча багов, но мы стараемся как можно больше доделывать проект. 
Также еще контроллер не готов, в скором времени выложим код и для него, но у нас и есть файл эмулятора для теста.
Веб интерфейс помогал писать ии и некоторые функции.
**Настройка**: Откройте файл login_db.py. Найдите функцию init_db и найдите там         
```if count == 0:
            default_users = [
                ('username', 'password', 'admin')
            ]
```

Замените username на свой юзернейм, а password на свой пароль.

Откройте web_Server.py, найдите там `app.secret_key = 'change_this'`. Измените значение change_this на сложный пароль.
Скачайте любой mqtt брокер и запустите его на том же компьютере. Запустите web_Server.py. Зайдите на {server ip}:80 и войдите в систему. Откройте в папке test файл
