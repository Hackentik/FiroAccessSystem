# Firo Access System

**Firo Access System** представляет собой открытую систему сетевого сервера контроля и управления доступом (СКУД), разработанную с целью предоставления возможности управления доступом, а также автоматического открытия дверей в соответствии с заранее установленным расписанием. В перспективе система будет поддерживать API.

На текущий момент проект находится в стадии разработки, и могут присутствовать определенные технические недоработки. Однако мы активно работаем над устранением выявленных недостатков и совершенствованием системы.

Контроллер системы еще не готов к использованию. В ближайшее время планируется публикация кода для контроллера, а также уже предоставлен файл эмулятора для тестирования.

Веб-интерфейс системы был разработан с использованием технологий искусственного интеллекта.

Настройка системы

Для настройки системы необходимо выполнить следующие шаги:

1. Открыть файл login_db.py.
2. Найти функцию init_db.
3. В функции init_db заменить строку:
```
if count == 0:
    default_users = [
        ('username', 'password', 'admin')
    ]
```
на:
```
if count == 0:
    default_users = [
        (ваш_юзернейм, ваш_пароль, 'admin')
    ]
```
4. Открыть файл web_Server.py.
5. Найти строку:

`app.secret_key = 'change_this'`

6. Заменить значение change_this на сложный пароль.
7. Скачать любой MQTT-брокер и запустить его на том же компьютере, где будет выполняться настройка системы.
8. Запустить файл web_Server.py.
9. Открыть веб-браузер и перейти по адресу {server ip}:80.
10. Войти в систему, используя указанные ранее учетные данные.
11. Открыть файл dooremulator.py в папке test на том же компьютере, где запущен сервер. Файл автоматически подключится к серверу.

Эти шаги обеспечат базовую настройку системы и позволят приступить к её дальнейшему использованию и тестированию.
